# PrintToolAI

PrintToolAI is a .NET 10.0 command-line utility.
It takes the arguments passed to the command line and seperates them with a comma.

This repo is created for demoing GitHub Model possibility and how this can help automate some scenario's.

## Getting Started

### Prerequisites
- [.NET 10.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
